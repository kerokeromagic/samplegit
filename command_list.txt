Git
Pull
	リモートリポジトリから最新の状態をローカルリポジトリに反映させる、サーバーからダウンロードしてアップデートするイメージ

Merge
	アップストリームブランチの状態をローカルブランチに反映させる、

作業場、実験場、本番環境
  本番環境のものを作業場に用意するのがpull、本番環境のものを実験場に持ってくるのがfetch、実験場から作業場に持ってくるのがmerge、

マージはコミットの逆、フェッチはプッシュの逆

プルリクエスト
	ローカルリポジトリの変更点を変更箇所をわかりやすくした状態で評価する人間に通知するgitの機能、

コンフリクト
	gitで管理する複数のデータに重複や競合が発生すること、gitから怒られる、データが異なるブランチのものをマージしようとすると発生するらしい、
  修正すべき箇所が自動でファイルに書き込まれるのでそこを残したい方残して修正

Checkout
	任意のブランチに移動できるコマンド、今どのブランチにいるかはbranchコマンド

Clone
	リモートリポジトリをローカルにコピーすること
Fork
	cloneと同じことだが、親リポジトリの所有者にforkしたことが通知される

Stash
	コミットしていない変更を一度退避させる(addによらない)
  Stash listで退避させた変更の一覧表示、stash applyで戻す
